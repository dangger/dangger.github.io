Microsoft SQL Server 2008安装备忘

4月 22, 2015

0：安装.NET Framework 3.5和Windows Installer 4.5 Redistributable，还有个OPENAL不知道有没有必要去安装。
1：打开安装包，一路下一步。
2：可能遇到的问题有:服务器配置那里不知道填写什么账号密码。
有人说是系统管理员账号，亲测无效。
我的解决办法是:
首先要点击“对所有SQL Server 服务使用相同的账号”，然后点击下拉菜单，选择localsystem。
上面下面都要这么写。
然后就可以一路下一步，记得需要添加当前账户管理员的时候添加一下。
3:漫长安装后成功。
